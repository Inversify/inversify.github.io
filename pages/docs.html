Docs

The InversifyJS features are docummented on the GitHub wiki. InversifyJS is powerful IoC container and supports the following features:

  • Support for classes
  • Support for Symbols
  • Container API
  • Declaring container modules
  • Container snapshots
  • Controlling the scope of the dependencies
  • Declaring optional dependencies
  • Injecting a constant or dynamic value
  • Injecting a class constructor
  • Injecting a Factory
  • Auto factory
  • Injecting a Provider (asynchronous Factory)
  • Activation handler
  • Middleware
  • Multi-injection
  • Tagged bindings
  • Create your own tag decorators
  • Named bindings
  • Default target
  • Support for hierarchical DI systems
  • Contextual bindings & @targetName
  • Property injection
  • Circular dependencies
  • Inheritance
